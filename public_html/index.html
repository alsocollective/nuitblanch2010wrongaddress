If you've tried to log into the network and have been uncessfull there is a few things that we can trouble shoot.

  • Try refreshing your page.
    On Android swipe down on the browser page.
    On Apple Safari browser swipe down and click the refresh icon next to the url.
    On Apple Chrome browser swipe down till the refresh icon highlights.

  • When you recive a prompt to login into the network? There is no log in, to continue from this page accecept using the network without internet.
    On Android when promted to login the top right of your screen should be a button, when clicked will promp you to "use this network as is".
    On Apple when promted to login click "Cancle" (top right), then click "Use Without Internet".

  • The device is not connecting to the network. Some Androids by default will not connect to 2.4ghz networks, this needs to be enabled on the device.
    On Android navigate to your "Settings" > "Wi-Fi" > "advance Wi-fi " (top right) > "Wi-Fi frequencey band", and set it to "Automatic".
    On Apple this will not effect you.

  • The network might be under a heavy load at the moment, please try reconnecting once inside the exhibition or in a few minutes again.
    There are multiple acces points thoughout the exhibition (all named the same), depending on your proximity to each access point you will connect to a different access point.
